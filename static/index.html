How to Use

Static Mode (Single Frame)

  1. Select "Static" detection mode
  2. Click "Start Camera" to enable your webcam
  3. Click "Start Live Detection" for continuous recognition
  4. Form ASL letters (A, B, C, etc.)
  5. Best for: Static hand poses

Temporal Mode (Motion-based)

  1. Select "Temporal" detection mode
  2. Start camera and live detection
  3. Perform dynamic gestures with movement
  4. System analyzes motion over time (~1 second)
  5. Best for: Dynamic signs like J, Z
